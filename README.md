# 🏠 House Price Prediction App

A complete end-to-end machine learning project that predicts California house prices using **Linear Regression** with a beautiful **Streamlit** dashboard.

## 📋 Project Description

This project uses the **California Housing dataset** (bundled with scikit-learn) to train a Linear Regression model that predicts the **median house value** for block groups across California.

The dataset contains **20,640 samples** with **8 features**:

| Feature      | Description                                          |
|-------------|------------------------------------------------------|
| MedInc      | Median income of households ($10k)                   |
| HouseAge    | Median age of houses (years)                         |
| AveRooms    | Average number of rooms per household                |
| AveBedrms   | Average number of bedrooms per household             |
| Population  | Block group population                               |
| AveOccup    | Average number of occupants per household            |
| Latitude    | Latitude coordinate                                  |
| Longitude   | Longitude coordinate                                 |

The **target variable** is the median house value in units of $100,000.

## 🚀 Getting Started

### 1. Clone / Navigate to the project

```bash
cd house-price-ml
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🧠 Train the Model

Run the training script to fetch the dataset, train the model, and save it:

```bash
python train.py
```

**Expected output:**

```
🏠 House Price Prediction — Model Training
----------------------------------------------
✅ Dataset loaded: 20640 samples, 8 features
   Train set: 16512 | Test set: 4128
✅ Model trained successfully

📊 Model Evaluation Metrics (Test Set)
==========================================
   R² Score            : 0.6063
   Mean Absolute Error : 0.5332  (in $100k units)
                       ≈ $53,322
==========================================

💾 Model saved to: model/model.pkl
```

The trained model pipeline is saved at `model/model.pkl`.

---

## 🖥️ Launch the Streamlit App

```bash
streamlit run app.py
```

This will open a browser window at `http://localhost:8501` with the interactive dashboard featuring:

- **Input form** for all 8 housing features
- **Live map** showing the selected location
- **Instant prediction** displayed as a dollar amount
- **Feature importance** chart (model coefficients)
- **Correlation heatmap** across all features
